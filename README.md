# Improving the Efficiency of IDS/IPS Systems in Detecting Network Layer Attacks based on Machine Learning Methods
## About the project
This is the Computer Engineering Project (CO4041).
Major: Computer Engineering - Ho Chi Minh City University of Technology.

## Brief info
### Background and reasons for choosing the topic

In the context of network systems increasingly becoming targets of sophisticated cyber attacks, ensuring information security is a big challenge. Traditional IDS/IPS systems often rely on fixed rules, which are easily bypassed by new attack techniques. The development of Machine Learning opens up a new approach, helping the system detect abnormal behaviors intelligently and adaptively.

### Research objectives

* Improve the efficiency of detecting network layer attacks by integrating machine learning into the IDS/IPS system.
* Propose a machine learning model suitable for real network data.
* Evaluate model performance through indicators such as Accuracy, Precision, Recall, F1-score.

### Research scope

* Focus on network layer attacks such as DoS, Port Scanning, Spoofing.
* Use public datasets such as CIC-IDS2017 or CSE-CIC-IDS2018.
* Do not delve into application layer attacks or malware.

### Approach

* Overview of machine learning techniques in intrusion detection.
* Analyze current models and suggest improvements.
* Experiment with the model on real data and evaluate the results.
